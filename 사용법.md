# 경쟁사 가격 조사 프로그램 사용법

## 실행
1. Python 최초 1회 설치 ("Add Python to PATH" 체크)
2. **미국 VPN(NordVPN) 켜기** ← 필수! (차단 사이트는 VPN 없으면 안 됨)
3. `run.bat` 더블클릭 → 자동 조사
   - 첫 실행: 필요한 부품(가벼운 패키지)만 자동 설치(최초 1회, 잠깐)
4. 끝나면 "결과 엑셀 열기"

## 동작 방식
- 우리 제품 목록과 수동보정·단위주의는 **구글시트**에서 자동으로 읽어옵니다.
- Cloudflare 차단 사이트(meamoshop 등)는 **curl_cffi(크롬 TLS 흉내)로 브라우저 없이** 통과합니다.
  (예전의 150MB 브라우저 엔진 설치는 더 이상 필요 없습니다.)
- "사이트 상태" 시트에서 각 사이트 정상/차단 여부를 확인할 수 있습니다.

## 안 될 때
- 특정 사이트가 계속 "차단됨" → 미국 VPN이 켜져 있는지 먼저 확인.
- 특정 사이트 결과가 0개면 알려주세요(카테고리 주소 조정이 필요할 수 있음).
